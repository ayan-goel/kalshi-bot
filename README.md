# Kalshi Trading Bot

Mid-frequency trading system for Kalshi focused on **market making + controlled event-driven trading**.

## Features
- Real-time market data + local order books  
- Fair value estimation for binary markets  
- Passive limit order execution  
- Inventory + risk management  
- PnL, fills, and position tracking  
- Demo + live trading support  

## Stack
- Rust (core trading + execution)  
- Python (research + backtesting)  
- Postgres (data + logs)  
- Redis (optional)  
- React dashboard  

## Core components
- Market Data  
- Strategy (fair value + quoting)  
- Execution  
- Risk
